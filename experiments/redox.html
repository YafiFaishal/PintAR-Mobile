Persamaan Reaksi:

Zn + Cu²⁺ → Zn²⁺ + Cu

Oksidasi (lepas e⁻)

Zn → Zn²⁺ + 2e⁻

E° = +0.76 V

Reduksi (terima e⁻)

Cu²⁺ + 2e⁻ → Cu

E° = +0.34 V

Bilangan Oksidasi:

Zn: 0 → +2 (oksidasi) | Cu: +2 → 0 (reduksi)

E° sel = E° katoda − E° anoda = +0.34 − (−0.76) = +1.10 V

✓ Reaksi spontan (E° > 0)
